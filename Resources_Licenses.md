# Open-Source Technology

This list includes the open-source Java/JavaScript libraries and resources utilized by SilkBuilder. Some are used for compiling the SilkBuilder classes, some are dependencies, and others are included to assist developers. All these resources were chosen because they have licenses that allow free use and distribution without restrictions on the code using them. This is intended to safeguard the privacy and ownership of the developerâ€™s code.

- [**Apache Commons**](https://commons.apache.org/) - A collection of reusable Java libraries under the Apache Software Foundation that provide utilities for common programming tasks. ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**Apache HttpComponents**](https://hc.apache.org/) - The Apache HttpComponents project is responsible for creating and maintaining a toolkit of low-level Java components focused on HTTP and associated protocols. ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**ASM**](https://asm.ow2.io/) - ASM is an all-purpose Java bytecode manipulation and analysis framework. It can be used to modify existing classes or to dynamically generate classes, directly in binary form. ([BSD V3 License](https://gitlab.ow2.org/asm/asm/-/blob/master/LICENSE.txt))
- [**Bootstrap**](https://getbootstrap.com/) - Powerful, extensible, and feature-packed frontend toolkit. Build and customize with Sass, utilize prebuilt grid system and components, and bring projects to life with powerful JavaScript plugins. The SilkBuilder components are built based on Bootstrap specifications and classes for easy theme building. ([MIT License](https://getbootstrap.com/docs/4.0/about/license/))
- [**ChartJS**](https://www.chartjs.org/) - Chart.js is an open-source JavaScript library for creating animated, responsive, and customizable charts and graphs on websites using the HTML5 canvas element. ([MIT License](https://github.com/sweetalert2/sweetalert2/blob/main/LICENSE))
- [**CK Editor V3**](https://ckeditor.com/blog/CKEditor-3.0-here/) - HTML WYSIWYG editor with countless features and wide browser compatibility. Although this is an older version, it still functions effectively and allows for free distribution version. (Mozilla Public License V1)
- [**CodeMirror** ](https://codemirror.net/)- CodeMirror is a code editor component for the web. It can be used in websites to implement a text input field with support for many editing features, and has a rich programming interface to allow further extension. SilkBuilder uses CodeMirror as its application code editor. ([MIT License](https://github.com/codemirror/dev?tab=License-1-ov-file))
- [**Commonmark Java**](https://github.com/commonmark/commonmark-java) - Java library for parsing and rendering Markdown text according to the CommonMark specification. ([BSD V2 License](https://github.com/commonmark/commonmark-java?tab=BSD-2-Clause-1-ov-file)) 
- [**Font Awesome**](https://fontawesome.com/) - Font Awesome is the Internet's icon library and toolkit, used by millions of designers, developers, and content creators. SilkBuilder comes loaded with Font Awesome, and the components utilize Font Awesome icons as part of their configuration. ([Font Awesome Free License](https://fontawesome.com/license/free))
- [**Jasypt**](https://github.com/jasypt/jasypt) - Jasypt is a Java library for encrypting and decrypting sensitive data with minimal effort, simplifying the process for developers without needing deep cryptography knowledge.  ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**JavaMail API**](https://javaee.github.io/javamail/) - The JavaMail API provides a platform-independent and protocol-independent framework to build mail and messaging applications. ([Oracle CDDL License](https://javaee.github.io/javamail/LICENSE))
- [**Jayway JsonPath**](https://github.com/json-path/JsonPath) - JSONPath is a query language for JSON data, analogous to XPath for XML. It provides a standardized way to navigate through and extract specific elements or sets of elements from a JSON document.  ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**JQuery**](https://jquery.com/) - jQuery is a fast, small, and feature-rich JavaScript library. It simplifies things like HTML document traversal and manipulation, event handling, animation, and Ajax with an easy-to-use API that works across many browsers. SilkBuilder uses JQuery to interact with HTML elements and execute Ajax calls. ([MIT License](https://jquery.com/license/))
- [**Json Simple**](https://code.google.com/archive/p/json-simple/) - JSON.simple is a lightweight Java toolkit designed for encoding and decoding JSON (JavaScript Object Notation) text. It provides a straightforward way to work with JSON data within Java applications. ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**J**](https://netplex.github.io/json-smart/)[**s**](https://github.com/netplex/json-smart-v2)[**on Smart**](https://netplex.github.io/json-smart/). Json-smart is a high-performance JSON (JavaScript Object Notation) processor library for Java. It is designed to be a fast and efficient alternative to other JSON parsing libraries, particularly known for its speed compared to libraries like json-simple. ([Apache 2 License](https://www.apache.org/licenses/LICENSE-2.0))
- [**jsoup**](https://jsoup.org/news/release-1.15.1) - jsoup is a Java library for working with real-world HTML. It provides a very convenient API for extracting and manipulating data, using the best of HTML5 DOM methods and CSS selectors. ([MIT License](https://jsoup.org/license))
- [**LangTags**](https://github.com/jsommers/langtags) - The purpose of langtags is to provide some simple mechanisms by which language tags can be parsed and validated. ([GPL-3.0 License](https://github.com/jsommers/langtags?tab=GPL-3.0-1-ov-file) ) 
- [**Spring Framework**](https://spring.io/projects/spring-framework) -The Spring Framework provides a comprehensive programming and configuration model for modern Java-based enterprise applications - on any deployment platform. SilkBuilder uses Spring Framework for database communication and MVC controllers to interact with JSP views. ([Apache 2 License, Version 2.0](https://springframework.net/license/))
- [**SweetAlert2**](https://sweetalert2.github.io/) - Provides a beautiful, responsive, customizable, and accessible replacement for Javascript's popup boxes. SilkBuilder utilizes SweetAlert2 to display dialog boxes, which can be directly configured in SilkBuilder components. ([MIT License](https://github.com/sweetalert2/sweetalert2/blob/main/LICENSE))
