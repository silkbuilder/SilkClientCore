** WARNING **

The folder WEB-INF/lib-ext should contain the extra
.jar files neded to add extra functionality to applications.
These are the .jar files  which are not part of
SILK requirements.

The WEB-INF/lib gets cleaned after .jar rebuild.
Any extra .jar file added directly to the WEB-INF/lib,
WEB-INF/lib-silk  and WEB-INF/lib-req folder will
be removed.
